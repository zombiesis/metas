# Form Specifications

Admissions fields: program selector, student name, parent name, phone, WhatsApp, email, city, current qualification, message, consent.

Contact fields: name, phone, email, inquiry type, message, consent.

Recruiter fields: company, contact person, designation, phone, email, hiring requirement, program interest, message, consent.

Alumni fields: name, graduation year, program, profession, company, phone, email, LinkedIn, message, consent.

Career fields: opening, applicant name, phone, email, qualification, experience, resume upload, cover letter, consent.

All forms require production anti-spam, rate limiting, consent logging, CSV export, CRM routing, and analytics.
